# BlenderBridgeScripts

Reserved for free Blender Python scripts shipped with Shugan Unity Tools.

Paid scripts (e.g. `shugan_autorig_feet.py`) are distributed separately via
Shugan's store and install into `Assets/! Shugan/!_Lab/Script/`.

The Blender Bridge tool can run any `.py` script — drag one into the
"Script" field at the top of the tool, or use Browse to pick one
from anywhere on disk.

Paid scripts shipped by Shugan use the `shugan_*` prefix
(e.g. `shugan_autorig_feet.py`) to keep ownership unambiguous in
mixed projects.
